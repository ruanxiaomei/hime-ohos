# 薄荷输入法

薄荷输入法是一款面向 HarmonyOS 的 Rime 中文输入法，提供拼音候选、中英文切换、方案切换、主题设置、剪贴板等常用能力。

## 环境要求

- HarmonyOS SDK 6.1.0（API 23）
- HarmonyOS 6.1+
